## Description
This application allows the user to enter a file extension and returns  a
description for that file type.

## Features
- Search file extensions using a dictionary
- Handles invalid and unknown extensions gracefully with a general message of unknown file

## Data Structure
- `Dictionary<string, string>`  to store file extensions and their
  descriptions as key-value pairs.

## Program Structure
- **FileExtension.cs**  
  Contains the dictionary and logic for searching file extensions.

- **Program.cs**  
  for user input and displays results to the console.

## How to Run
1. Open the project in Visual Studio or JetBrains Rider
2. Build and run the application
3. Enter a file extension when prompted

- C#
- .NET Console Application
